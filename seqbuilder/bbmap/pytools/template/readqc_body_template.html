Insert Size

  • BBTools bbmerge.sh is used to perform this analysis. bbmerge merges paired reads into single reads by overlap detection. With sufficient coverage, bbmerge can also merge nonoverlapping reads using gapped kmers.
  • The "percentage reads joined" is calculated by (number of joined reads/total number of reads * 100).
  • Raw reads are used for the insert size calculation (no trimming or filtering).
